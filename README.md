# 功能点：
比较核心的几个点：
1. Nuxt动态分配路由：解决了Nuxt默认文件命名下路由动态分配的问题，并向社区提出了bug
2. 后端解析文章，文件流，事务，前端解析目录，树形递归，动态定位高度。
3. 日志功能：文件流解析日志，websocket推送日志，log4js生成日志，chokidar监控，考虑使用Radis
4. 管理页面：组件化表格，
5. 全局消息组件：fetch拦截器错误提示

登录页：
1. jwt+cookie策略
2. 登陆注册校验双向(前后端)
3. 一些细节上的实现：比如重复登陆，登陆清除(待取消后退)，记住密码等

其他：
1. 全局消息组件，响应错误和成功提示
